### Dependancies/Requirements to run code bundle:
```Python 3, Pygal, Lxml, PyGithub```


#### Instructions
1. Clone repository 
2. Open command prompt or terminal and cd into the Github-Visualisation folder
3. Use the command ```pip install -r requirements.txt``` to install all necessary dependancies 
4. Use the command ```python visualisation.py``` to run the python script
5. Enter a valid Github Username and a valid Github OAuth Token to prevent ratelimiting
6. After entering the username and token the program should run and create the visualisations on your browser for you
7. If you have any issues please contact me at ```alabraha@tcd.ie```.
